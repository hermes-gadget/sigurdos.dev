# sigurdos.dev

Source for the [SigurdOS firmware](https://github.com/hermes-gadget/SigurdOS-tdeck) website.

## Stack

- Static HTML + CSS
- [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) / [Pixelify Sans](https://fonts.google.com/specimen/Pixelify+Sans) from Google Fonts
- Zero dependencies. No build step.

## Deploy

The site is served via Nginx on the sigurdos.dev VM. Push to `main` and pull on the server.

Features

SlopOS packs a full mesh communication stack into a pocket-sized device. Every feature is built from the ground up for off-grid use.

💬

Multi-Channel Chat

Full Discord-style messaging with dynamic channel discovery, DM support, an emoji picker, and message history that survives power loss. Send text across your mesh network to individuals or channels — all with end-to-end encryption.

🗺️

Offline Tile Maps

Navigate with full offline JPEG tile maps stored on the SD card. Pan, zoom, and explore without any network connection. Tile downloader for 18 UK city presets included. Rendered directly to the display via the ESP32 hardware JPEG decoder.

🌍

Multi-Hop LoRa Mesh

Built on MeshCore, a battle-tested LoRa mesh networking library. Automatic route discovery, flood routing with ACKs, path-based direct messaging, and node discovery via periodic adverts. Every packet is encrypted with Ed25519 keys.

🔍

Network Visibility

Five dedicated screens for full mesh awareness: Heard (searchable RSSI list with columns), Network (recently active nodes), Trace (path quality to any contact), Signal (live RSSI/SNR with noise floor), and an integrated diagnostics terminal with colored log output.

⌨️

Hardware Keyboard + Trackball

Full QWERTY input via the I2C keyboard MCU with configurable backlight brightness. Five-direction trackball with debounced events for one-handed navigation through the entire UI. Touch input also supported via the GT911 capacitive controller.

⚙️

Live Configuration

On-device setup wizard for first boot, live radio config (frequency, spreading factor, TX power, coding rate), date/time setting, keyboard backlight control, node naming, and channel management with hashtag or PSK-based joining. All settings persist in NVS.

📱

Flutter Mobile Client

Connect your phone or desktop to the mesh via BLE, USB, or TCP using the SlopOS Client — a full Flutter app with the same pixel theme. Read messages, manage channels, and browse contacts without the T-Deck display.

🌍

GPS + Battery Management

Built-in GPS support with multi-constellation NMEA parsing ($GP and $GN), battery voltage monitoring with percentage calculation, deep sleep on low battery, and automatic display auto-off after 30 seconds of inactivity.

💻

Developer Friendly

PlatformIO-based build system, 172+ native unit tests with hardware mocks, CI/CD via GitHub Actions, CONTRIBUTING.md, and a web flasher for zero-tool firmware installation. Everything is GPL-3.0 and open for contributions.

Map Tiles ALPHA

SlopOS supports offline tile maps on the T-Deck. Generate tiles right in your browser and download them as a zip — no Python scripts, no server-side processing. Just pan to your area, select zoom levels, and download.

SD Card Path

Copy the extracted tiles/ folder to the root of your T-Deck's SD card. Structure: tiles/zoom/x/y.jpg. This also works with Ripple firmware tiles.

Zoom Levels

Higher zoom = more detail but more tiles. A single zoom 10 tile covers ~38km², zoom 14 covers ~2.5km². For an average city, zooms 10–14 give a good balance of detail and file size. Each tile is 256×256 PNG.

Data Usage

Tiles are downloaded from OpenStreetMap's tile servers. For a city-sized area: zooms 10–14 = ~10–50 MB. Zooms 10–16 = ~50–200 MB. Each tile is ~10–50 KB. Generate on WiFi before heading out.

Tile Generator

Pan and zoom to your area of interest, then click “Download Current View”. The tool downloads tiles from OpenStreetMap for the visible area at the selected zoom range.

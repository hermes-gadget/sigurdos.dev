About

SlopOS is a fully open source firmware ecosystem built on top of MeshCore, a lightweight LoRa mesh networking library. It transforms the LilyGo T-Deck into a portable mesh communication device with a full graphical interface.

No towers. No SIM cards. No internet required. Just radios talking to radios.

Features

💬

Chat

Multi-channel chat with DM support, emoji picker, and channel history that survives power loss.

🗺

Offline Maps

JPEG tile-based offline maps from SD card. Pan, zoom, and navigate without any network.

🌍

Mesh Network

Multi-hop LoRa mesh with automatic routing, discovery, and end-to-end encryption.

🔍

Node Discovery

Heard, Network, Trace, and Signal screens give full visibility into mesh topology and signal quality.

⌨

Hardware Keyboard

Full I2C keyboard input with backlight control. Trackball for one-handed navigation.
